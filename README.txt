Este projeto envolveu a criação de uma aplicação simples de Lista de Tarefas. Desenvolvi a interface usando HTML e a lógica de manipulação dos itens da lista com JavaScript. Foi uma prática para entender a interação entre elementos HTML, eventos e funções JavaScript.


This project involved creating a simple To-Do List application. I developed the interface using HTML and the logic for manipulating the list items using JavaScript. It was a practice to understand the interaction between HTML elements, events, and JavaScript functions.
